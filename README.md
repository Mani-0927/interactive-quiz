# Interactive Quiz

Hard

## Technologies Used

## Project Overview
This project is a web-based Interactive Quiz application developed using HTML, CSS, and JavaScript.  
It allows users to answer multiple-choice questions and see their score at the end.  
The quiz is fully responsive and user-friendly.

## Project Description
The Interactive Quiz provides multiple-choice questions and calculates the score based on correct answers.  
It gives visual feedback for correct and incorrect answers and prevents submission if any question is unanswered.

## Features
- Multiple-choice questions (at least 5)  
- Select answers using radio buttons  
- Visual feedback: correct (green) and wrong (red)  
- Score displayed after submission  
- Prevents submission if questions are unanswered  
- Responsive design for desktop, tablet, and mobile  
- Beginner-friendly and easy to understand

## How to Use
1. Download and extract the project folder.  
2. Open `index.html` in any modern web browser.  
3. Select answers for each question.  
4. Click the **Submit Quiz** button.  
5. See your score and which answers were correct or wrong.
